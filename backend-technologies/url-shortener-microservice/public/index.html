Shortener Microservice

Enter a URL and get an URL you can share


Usage:

Endpoint (POST request)

{url}/api/shorturl

Example body


{
    "url": "https://www.asterki.ga/"
}

Example response


{ 
    "original_url": "https://freeCodeCamp.org",
    "short_url": 1
}


Endpoint (GET request)

{url}/api/shorturl/:urlID

Example URL

{url}/api/shorturl/2
